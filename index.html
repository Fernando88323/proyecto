Fuentes Personalizadas

Uno de los componentes referidos a fuentes en CSS3 es la regla font-face. Esta regla permite tener un mayor control en las tipografías ya que se puede añadir directamente nuevas fuentes en el documento usando reglas CSS.

CSS Grid

El módulo de diseño de cuadrícula CSS ofrece un sistema de diseño basado en cuadrícula, con filas y columnas, lo que facilita el diseño de páginas web sin tener que usar flotantes y posicionamiento.

Fundamentos de JavaScript

JavaScript es el lenguaje de programación que debes usar para añadir características interactivas a tu sitio web.
